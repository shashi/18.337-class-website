Introduction

Have you ever wondered why:

  • code that is easy to write is never high-performance and
  • high-performance code is never easy to write?

In this course we'll break down all the barriers that get in the way of writing high performance code. We shall cover topics in machine learning, numerical linear algebra, computer science and performance engineering.

Also watch out for exciting guest lectures!

Getting Started

  1. Create an account on GitHub and add your SSH key
  2. Fillup the signup form.
  3. Signup onto Piazza.
  4. Install Julia 1.0 on your laptop (platform specific instructions)
  5. We recommend using Julia via Juno, VSCode or Jupyter
